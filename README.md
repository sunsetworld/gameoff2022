# gameoff2022
 Entry for Game Off 2022.


 
 Barrelboy 2022.

-- Download Instructions --

Windows.

Unzip the folder and run the gameoff22 .exe file.

Steam Deck/Linux.

Download and unzip the folder to a favourite place of your choosing.
Go to Steam, open the library and click "ADD A GAME", and then Add a Non-Steam Game. 
Once the "Add a Game" dialogue appears, click "BROWSE..." select the file type dropdown and select "All Files."
Locate the game by browsing to the folder where the game is saved and select the gameoff22.exe file and select "OPEN."
Select "ADD SELECTED PROGRAMS."
Find the game in your library. Select the cog icon to the right. Select "Properties...", then "COMPATIBILITY",
and tick the box next to "Force the use of a specific Steam Play compatibility tool.", 
ensure that the box below says "Proton Experimental."
(Optional, but still recommended) Go back to "SHORTCUT" and change the name of the game.
NB: Performance on Steam Deck is expected to be between 20-30fps, due to the usage of advanced next gen features. 

-- Controls --
WASD/Left Thumbstick: Move player.
Mouse/Arrow Keys/Right Thumbstick: Move camera.
Shirt: Sprint.
Left Mouse/Space Bar: Jump.
1/Left Thumbstick Click: Toggle between first and third person.
P/Esc/Gamepad Top Left: Pause.
Q: Quit.
